## ⚔️ Compiled vs Interpreted — เจาะลึกต่างกันยังไง?

```
   ┌──────────────────────────────────────────────────────────────────────┐
   │                                                                      │
   │   Compiled (C, C++, Rust, Go)        Interpreted (Python, JS, Ruby) │
   │                                                                      │
   │   ┌──────┐   ┌──────────────┐        ┌──────┐   ┌──────────────┐   │
   │   │Source│──▶│   Compiler   │        │Source│──▶│  Interpreter │   │
   │   │ Code │   │ (แปลทั้งหมด)  │        │ Code │   │ (แปลทีละบรรทัด)│  │
   │   └──────┘   └──────┬───────┘        └──────┘   └──────┬───────┘   │
   │                     │                                   │           │
   │                     ▼                                   ▼           │
   │              ┌──────────────┐                    ┌──────────────┐   │
   │              │ Machine Code │                    │  รันได้ทันที  │   │
   │              │ (เร็วมาก!)   │                    │  (ช้ากว่า)    │   │
   │              │ .exe / .bin  │                    │              │   │
   │              └──────────────┘                    └──────────────┘   │
   │                                                                      │
   │   ✅ เร็วกว่ามาก                      ✅ แก้บั๊กง่ายกว่า              │
   │   ❌ ต้อง compile ใหม่ทุกครั้งที่แก้   ❌ ช้ากว่า                     │
   │   ❌ ย้ายข้าม OS ไม่ได้               ✅ รันทุก OS เลย              │
   │                                                                      │
   │   💡 เปรียบเทียบ:                                                     │
   │      Compiled = แปลหนังสือทั้งเล่ม แล้วค่อยขาย                          │
   │      Interpreted = แปลทีละหน้า ขายได้ทันที                            │
   │                                                                      │
   └──────────────────────────────────────────────────────────────────────┘
```

## 📝 เขียนไฟล์ Python แรกของคุณ

```
   ขั้นตอน:

   1. เปิด Text Editor (VS Code / Notepad)
   2. สร้างไฟล์ใหม่ ตั้งชื่อ hello.py
   3. เขียนโค้ดด้านล่าง
   4. บันทึกไฟล์
   5. เปิด Terminal → พิมพ์: python hello.py
```

```python
# ไฟล์: hello.py
# นี่คือ Comment — Python จะไม่รันบรรทัดนี้

print("=" * 40)
print("  🐍 สวัสดีชาว Python!")
print("=" * 40)

name = "มือใหม่"
lesson = 1

print(f"สวัสดี {name}!")
print(f"คุณกำลังเรียนบทที่ {lesson}")
print(f"Python เวอร์ชัน: มากกว่า 3.6")

# ลองคำนวณ
a = 10
b = 3
print(f"\n{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} * {b} = {a * b}")
print(f"{a} / {b} = {a / b:.2f}")
print(f"{a} // {b} = {a // b}  (หารปัดเศษ)")
print(f"{a} % {b} = {a % b}   (เศษ)")
print(f"{a} ** {b} = {a ** b}  (ยกกำลัง)")
```

```
   รันคำสั่ง: python hello.py

   Output:
   ┌─────────────────────────────────────────┐
   │  ========================================│
   │    🐍 สวัสดีชาว Python!                  │
   │  ========================================│
   │  สวัสดี มือใหม่!                          │
   │  คุณกำลังเรียนบทที่ 1                     │
   │  Python เวอร์ชัน: มากกว่า 3.6             │
   │                                         │
   │  10 + 3 = 13                            │
   │  10 - 3 = 7                             │
   │  10 * 3 = 30                            │
   │  10 / 3 = 3.33                          │
   │  10 // 3 = 3  (หารปัดเศษ)              │
   │  10 % 3 = 1   (เศษ)                    │
   │  10 ** 3 = 1000  (ยกกำลัง)              │
   └─────────────────────────────────────────┘
```

## 🔑 สิ่งสำคัญที่ต้องรู้ก่อนไปบทต่อไป

```
   ┌──────────────────────────────────────────────────────────────────┐
   │                                                                  │
   │   1️⃣  Python เป็น Interpreted Language                           │
   │      → แปลทีละบรรทัด รันได้ทันที ไม่ต้อง compile                 │
   │                                                                  │
   │   2️⃣  Python ใช้ Indent (ย่อหน้า) แทน { }                       │
   │      → โค้ดที่ย่อเข้าไป = อยู่ในกลุ่มเดียวกัน                      │
   │                                                                  │
   │   3️⃣  Python เป็น Case-Sensitive                                │
   │      → Name ≠ name ≠ NAME (ตัวเล็ก-ใหญ่ ต่างกัน!)                │
   │                                                                  │
   │   4️⃣  Comment ใช้ #                                             │
   │      → ทุกอย่างหลัง # Python จะไม่รัน                             │
   │                                                                  │
   │   5️⃣  ไม่ต้องใช้ ; ปิดท้ายบรรทัด                                  │
   │      → ต่างจาก Java/C ที่ต้องใช้ ;                                │
   │                                                                  │
   └──────────────────────────────────────────────────────────────────┘
```
