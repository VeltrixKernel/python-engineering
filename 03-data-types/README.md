## 1. Numeric Types (ตัวเลข)

```
   ┌─────────────────────────────────────────────────────────────┐
   │                    Numeric Types                            │
   │                                                             │
   │   ┌──────────┐  ┌──────────┐  ┌──────────┐                │
   │   │   int    │  │  float   │  │ complex  │                │
   │   │ จำนวนเต็ม│  │ ทศนิยม   │  │ จำนวนเชิง│                │
   │   │          │  │          │  │ ซ้อน     │                │
   │   │ 42       │  │ 3.14     │  │ 3 + 4j   │                │
   │   │ -7       │  │ -0.001   │  │ 2 - 1j   │                │
   │   │ 1000000  │  │ 1.5e10   │  │          │                │
   │   └──────────┘  └──────────┘  └──────────┘                │
   │                                                             │
   └─────────────────────────────────────────────────────────────┘
```

```python
# int - จำนวนเต็ม
age = 25
count = -100
big_num = 1_000_000     # ใช้ _ ช่วยอ่านง่าย

# float - ทศนิยม
pi = 3.14159
temperature = -15.5

# complex - จำนวนเชิงซ้อน
z = 3 + 4j
```

```
   ⚠️ ข้อระวังเรื่อง float:

   ┌──────────────────────────────────────────────────┐
   │                                                  │
   │   >>> 0.1 + 0.2                                  │
   │   0.30000000000000004   ← ไม่ใช่ 0.3 พอดี!       │
   │                                                  │
   │   เพราะคอมพิวเตอร์เก็บเลขเป็นฐาน 2 (binary)     │
   │   บางเลขทศนิยมเก็บเป็น binary ไม่ตัว             │
   │                                                  │
   │   วิธีแก้: ใช้ round() หรือ decimal module        │
   │   >>> round(0.1 + 0.2, 1)                        │
   │   0.3                                            │
   │                                                  │
   └──────────────────────────────────────────────────┘
```

## 2. String (ข้อความ)

```python
name = "สมชาย"
greeting = 'Hello'
multi_line = """บรรทัดที่ 1
บรรทัดที่ 2
บรรทัดที่ 3"""
```

```
   String ในหน่วยความจำ:

   name = "Python"

   ┌─────┬─────┬─────┬─────┬─────┬─────┐
   │  P  │  y  │  t  │  h  │  o  │  n  │
   └──┬──┴──┬──┴──┬──┴──┬──┴──┬──┴──┬──┘
      │     │     │     │     │     │
      0     1     2     3     4     5   ← Index จากซ้าย
     -6    -5    -4    -3    -2    -1   ← Index จากขวา
```

```
   String Methods ที่ใช้บ่อย:

   ┌─────────────────────────────────────────────────────────┐
   │                                                         │
   │   "Hello".upper()      →  "HELLO"       ตัวใหญ่ทั้งหมด  │
   │   "Hello".lower()      →  "hello"       ตัวเล็กทั้งหมด  │
   │   "  hi  ".strip()     →  "hi"          ตัด space หัวท้าย│
   │   "a,b,c".split(",")   →  ["a","b","c"]  แยก string    │
   │   "-".join(["a","b"])  →  "a-b"         รวม string    │
   │   "hello".replace("l","r") → "herro"    แทนที่         │
   │   "hello".find("ll")   →  2             หาตำแหน่ง      │
   │   "hello".count("l")   →  2             นับ            │
   │                                                         │
   └─────────────────────────────────────────────────────────┘
```

```python
# f-string (แนะนำสุด!)
name = "สมชาย"
age = 25
print(f"ชื่อ: {name}, อายุ: {age}")

# Output: ชื่อ: สมชาย, อายุ: 25
```

## Type Conversion (แปลงประเภท)

```
   ┌─────────────────────────────────────────────────────────────┐
   │                                                             │
   │   int("42")      →  42          string → integer            │
   │   float("3.14")  →  3.14        string → float              │
   │   str(42)        →  "42"        number → string             │
   │   bool(1)        →  True        number → boolean            │
   │   bool(0)        →  False       0 = False                   │
   │   bool("")       →  False       string ว่าง = False         │
   │   bool("hello")  →  True        string มีค่า = True         │
   │                                                             │
   │   ⚠️  แปลงไม่ได้จะเกิด Error:                               │
   │   int("hello")   →  ValueError!                             │
   │                                                             │
   └─────────────────────────────────────────────────────────────┘
```

## ตรวจสอบประเภท

```python
type(42)        # <class 'int'>
type(3.14)      # <class 'float'>
type("hello")   # <class 'str'>
type(True)      # <class 'bool'>
type(None)      # <class 'NoneType'>

# ตรวจสอบว่าเป็นประเภทนี้หรือเปล่า
isinstance(42, int)       # True
isinstance("hi", str)     # True
isinstance(3.14, int)     # False
```

```
   ┌──────────────────────────────────────────────────────────────┐
   │                  สรุป Data Types                             │
   │                                                              │
   │   int      →  จำนวนเต็ม         42, -7, 1_000_000            │
   │   float    →  ทศนิยม            3.14, -0.001                 │
   │   str      →  ข้อความ           "Hello", 'World'              │
   │   bool     →  True/False        True, False                   │
   │   NoneType →  ไม่มีค่า           None                          │
   │                                                              │
   │   🔑 สำคัญ:                                                  │
   │   ├── ใช้ type() เช็คประเภท                                  │
   │   ├── ใช้ isinstance() เช็คแบบปลอดภัยกว่า                   │
   │   ├── float ไม่แม่นยำ 100% (0.1+0.2 ≠ 0.3)                  │
   │   ├── None ≠ 0 ≠ "" ≠ False                                  │
   │   └── f-string คือวิธี format string ที่ดีที่สุด              │
   │                                                              │
   │   ต่อไป: บทที่ 4 — Operators ➕                              │
```
