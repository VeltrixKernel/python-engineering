## ตัวแปรใน Python เป็นอย่างไร?

```python
# ประกาศตัวแปรง่าย ๆ
name = "สมชาย"
age = 25
height = 175.5
is_student = True
```

```
   ┌─────────────────────────────────────────────────────────────┐
   │                       RAM (หน่วยความจำ)                     │
   │                                                             │
   │   Address    │  Variable  │  Value       │  Type           │
   │   ───────────┼────────────┼──────────────┼────────────     │
   │   0x1000     │  name      │  "สมชาย"     │  str            │
   │   0x1008     │  age       │  25          │  int            │
   │   0x1010     │  height    │  175.5       │  float          │
   │   0x1018     │  is_student│  True        │  bool           │
   │                                                             │
   └─────────────────────────────────────────────────────────────┘
```

## การตั้งชื่อตัวแปร

```
   ┌──────────────────────────────────────────────────────────────┐
   │                    กฎการตั้งชื่อตัวแปร                        │
   │                                                              │
   │   ✅ ใช้ได้:                                                  │
   │   ┌────────────────────────────────────────────────────┐     │
   │   │  name          ← ตัวอักษร + ตัวอักษร               │     │
   │   │  user_name     ← snake_case (แนะนำ!)              │     │
   │   │  age2          ← ตัวอักษร + ตัวเลข                 │     │
   │   │  _private      ← ขึ้นต้น _ (private)              │     │
   │   │  __dunder__    ← ขึ้นต้น __ (special method)      │     │
   │   └────────────────────────────────────────────────────┘     │
   │                                                              │
   │   ❌ ใช้ไม่ได้:                                               │
   │   ┌────────────────────────────────────────────────────┐     │
   │   │  2name          ← ขึ้นต้นด้วยตัวเลข                │     │
   │   │  my-name        ← มี - (ขีด)                      │     │
   │   │  my name        ← มี space (เว้นวรรค)             │     │
   │   │  class          ← เป็น keyword ของ Python         │     │
   │   └────────────────────────────────────────────────────┘     │
   │                                                              │
   │   🔑 แนะนำ: ใช้ snake_case เสมอ                             │
   │      user_name, total_score, is_active                       │
   │                                                              │
   └──────────────────────────────────────────────────────────────┘
```

## การกำหนดค่าหลายตัวแปร

```python
# วิธีที่ 1: ทีละตัว
x = 10
y = 20
z = 30

# วิธีที่ 2: กำหนดพร้อมกัน
x, y, z = 10, 20, 30

# วิธีที่ 3: ค่าเดียวกัน
x = y = z = 0
```

```
   วิธีที่ 2: x, y, z = 10, 20, 30

   ┌─────┐     ┌─────┐     ┌─────┐
   │ x=10│     │ y=20│     │ z=30│
   └─────┘     └─────┘     └─────┘

   วิธีที่ 3: x = y = z = 0

   ┌─────┐
   │ x=0 │◄──┐
   └─────┘   │
   ┌─────┐   │  ชี้ทุกตัว
   │ y=0 │◄──┤  ไปที่ค่า
   └─────┘   │  เดียวกัน
   ┌─────┐   │
   │ z=0 │◄──┘
   └─────┘
```

## การสลับค่า (Swap)

```python
a = 5
b = 10

# Python สลับค่าได้ง่ายมาก!
a, b = b, a
```

```
   ก่อน swap:                    หลัง swap:

   ┌─────────┐  ┌─────────┐     ┌──────────┐  ┌─────────┐
   │ a = 5   │  │ b = 10  │     │ a = 10   │  │ b = 5   │
   └─────────┘  └─────────┘     └──────────┘  └─────────┘

   ในภาษาอื่นต้องใช้ temp:

   ┌──────┐     ┌──────┐     ┌──────┐
   │temp=a│     │ a=b  │     │ b=temp│
   └──────┘     └──────┘     └──────┘
      3 ขั้นตอน!        vs     1 บรรทัดใน Python!
```

## ตัวอย่างจริง: คำนวณพื้นที่วงกลม

```python
PI = 3.14159
radius = 7

area = PI * radius ** 2

print(f"รัศมี: {radius}")
print(f"พื้นที่วงกลม: {area:.2f}")
```

```
   Output:
   ┌──────────────────────────────┐
   │  รัศมี: 7                    │
   │  พื้นที่วงกลม: 153.94        │
   └──────────────────────────────┘

   การคำนวณ:
   ┌──────────────────────────────┐
   │                              │
   │   area = PI × radius²        │
   │   area = 3.14159 × 7²        │
   │   area = 3.14159 × 49        │
   │   area = 153.93791           │
   │   area ≈ 153.94              │
   │                              │
   └──────────────────────────────┘
```
